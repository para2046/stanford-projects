# Stanford Projects

A collection of coursework and research projects from Stanford University.

## Projects

### [CS231N — NavLM v2](cs231n-navlm/)
**Visual Walking Directions Without a Compass** (Spring 2026)

End-to-end pipeline for training a compass-free vision-language navigation model on Zurich walking-tour videos. Fine-tuned Qwen 2.5 VL 7B achieves 67.2% accuracy on compass-free navigation from a 30.2% zero-shot baseline.

### [CS224N — Coding Agent SFT](cs224n-coding-agent/)
**Teaching Small Models to Code via BattleSnake** (Winter 2026)

Supervised fine-tuning of Qwen3-Coder-30B on coding agent trajectories from CodeClash BattleSnake. Compares Vanilla SFT, ReAct SFT, and Trajectory-Quality SFT using trajectories from 8 frontier AI teachers.
